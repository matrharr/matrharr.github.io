FirstAid

Role: Full Stack Engineer

Stack: Rails, jQuery, D3, Twilio, ActiveMailer, Devise, Stripe, PostgreSQL

First-Aid is a Rails app that gives users the ability to donate directly to local charities when people are displaced.The application uses the USGS and Google Maps APIs to correlate longitude and latitude and notifies the user through ActiveMailer and Twilio messages. Authentication is made secure through Devise and donations are made using Stripe payments. The user can dynamically view their personal donation data through D3 visualizations

Foundit

Role: Full Stack Engineer

Stack: Rails, jQuery, SQL, Paperclip, Google Maps API

Foundit is a Rails app that acts as a city-wide lost and found. It uses a search algorithm that compares descriptions of items based on cosine similarity and returns a ranked list of found items that match the lost item's description. A user may also search based on their previous location using Google Maps API. The site can be navigated dynamically with jQuery and Ajax as well as allowing the ability for user photo upload with Paperclip.
