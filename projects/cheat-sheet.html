SQL

Create Table

CREATE TABLE table_name(column1 DATA_TYPE, column2 DATA_TYPE, column3 DATA_TYPE);

Clauses

SELECT: Fetch data
UPDATE: Edit table data
WHERE: Used to extract those records that fulfill a specified criterion.
INSERT INTO: Used to insert new records into a table.
ALTER TABLE(table name) ADD COLUMN(new column name) DATA_TYPE: Add columns

Aggregate Functions

GROUP BY...DESC/ASC: Orders output
SUM: Returns sum of values.
MAX: Returns largest value.
MIN: Returns smallest value.
AVG: Returns average value.

Multiple Tables

Primary Key: Unique identifier, no null values.
Foreign Key: Contains primary key of a different table, can be null, not unique.
JOIN: Combine rows from different tables.
AS: Rename a column or table with alias.
